This is a full-stack web application developed as part of the **CEF345: Software Development Tools** course at the **Faculty of Engineering and Technology (FET), University of Buea**. 

The project aims to manage daily tasks efficiently while demonstrating the practical application of modern software development life cycle (SDLC) tools.

## üõ† Tech Stack & Tools
We have selected a modern stack to ensure scalability and environment consistency:

* **Project Management:** **Taiga.io** using the Scrum framework to manage our backlog and sprints.
***Version Control:** **Git & GitHub** for centralized code management and collaborative branching workflows.
***Frontend:** HTML5, CSS3, and JavaScript (with **React.js** integration planned).
***Backend:** **Node.js** with the **Express** framework.
***Database:** **MySQL** for relational data persistence.
***IDE:** **Visual Studio Code (VS Code)** standardized with relevant extensions for debugging.
* **DevOps & Containerization:** **Docker** and **Docker-Compose** to containerize the application and database environments.
